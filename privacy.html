Privacy Policy

This application connects TikTok accounts to n8n automation workflows for analytics and reporting purposes.

Data We Collect

  • TikTok profile information (username, ID, account details)
  • Video performance analytics (views, likes, comments, engagement metrics)
  • OAuth authentication tokens (for secure API access)

How We Use Data

Data is used only to generate analytics insights and automate workflows through n8n. No data is used for advertising or unrelated purposes.

Data Storage & Retention

Data is processed through automation workflows and may be temporarily stored in connected services (such as databases or Notion) controlled by the user. Data is retained only as long as needed for automation purposes.

Data Sharing

We do not sell, rent, or share personal data with third parties. Data is only processed through TikTok APIs and n8n automation workflows.

User Rights

Users can revoke TikTok access at any time through their TikTok account settings, which will immediately stop data access.

Security

We use secure API authentication (OAuth) and trusted automation infrastructure to protect data during processing.
